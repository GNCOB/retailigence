h1. Retailigence

A Ruby gem wrapper for interacting with the Retailigence API.

h2. Install

@gem install retailigence@

h2. Configure

@Retailigence.configure { |config| config.apikey = "your-api-key" }@

h2. Use

@results = Retailigence::Product.search(barcode: '001122334455', requestorid: '00deadbeef00', latitude: 27.92, longitude: -82.48)@

Results:

@product = results["RetailigenceSearchResult"]["Results"]["Result"][0]@
@name = product["Name"]@
@msrp = product["MSRP"]@
